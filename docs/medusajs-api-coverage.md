## 6. Mobile App Feature APIs (Custom)

| Feature / Page         | API Endpoint                | Method | Purpose / Usage                                      |
|------------------------|-----------------------------|--------|------------------------------------------------------|
| Push Notifications     | /store/mobile/notifications | POST   | Send a push notification to a user                  |
|                        | /store/mobile/notifications | GET    | List all notifications for a user                   |
| App Onboarding         | /store/mobile/onboarding    | GET    | Fetch onboarding content for the app                |
| Mobile Home Screen     | /store/mobile/home          | GET    | Fetch data for the mobile app home screen           |
| Mobile Profile         | /store/mobile/profile       | GET    | Fetch user profile optimized for mobile             |
| App Analytics          | /store/mobile/analytics     | POST   | Log app-specific events (e.g., screen views)        |
